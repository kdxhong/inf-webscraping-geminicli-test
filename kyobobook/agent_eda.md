주어진 데이터 파일(CSV, Excel 등)을 불러와 탐색적 데이터 분석(EDA)을 수행해주세요.

kyobobook/data/raw/kyobo_bestseller_api_20260117.csv

요구 사항:

1. 데이터 불러오기
   - CSV, XLSX, XLS, TSV 등 일반적인 데이터 파일 형식을 자동으로 인식하여 불러오기
   - pandas를 사용해 DataFrame(df) 형태로 로드
   - 파일 경로는 사용자 입력값으로 처리
   - 로드 과정에서 인코딩 문제나 시트 선택(Excel의 경우)을 자동으로 처리

2. 데이터 구조 및 품질 진단
   - info() 출력
   - 행(row)과 열(column) 개수, 컬럼 이름, 데이터 타입(dtype) 출력
   - 결측치(null) 개수 및 비율, 중복 데이터 존재 여부 확인

3. 기술통계 분석
   - 수치형 변수: 평균, 표준편차, 최소값, 최대값, 분위수(25%, 50%, 75%)
   - 범주형 변수: 고유값 개수, 최빈값, 각 범주의 빈도 및 비율

4. 결측치 및 이상치 탐색
   - 각 컬럼의 결측치 비율을 시각화 (bar plot)
   - 이상치(outlier)를 boxplot으로 탐지 및 시각화

5. 변수 간 관계 분석
   - 수치형 변수 간 상관계수 계산 및 히트맵 시각화
   - 범주형 ↔ 수치형 변수 간 관계 분석 (boxplot, barplot 등)

6. 데이터 분포 시각화
   - 주요 수치형 변수의 분포(histogram, kdeplot 등)
   - 범주형 변수의 비율(bar plot 등)

7. 분석 요약 및 인사이트
   - 데이터의 전반적 특성 요약
   - 주요 패턴, 이상치, 결측치, 불균형, 상관관계 등에서 얻은 인사이트를 한글로 설명

출력 및 저장 형식:

- **그래프 저장:**
  - 모든 시각화 결과를 `eda_images` 폴더에 PNG 파일로 저장
  - 모든 막대그래프의 경우 교차표나 피봇테이블로 해당 그래프 내용을 출력할 것
  - 폴더가 없을 경우 자동 생성
  - 파일명은 분석 목적이 드러나게 지정 (예: `missing_values.png`, `correlation_heatmap.png`, `distribution_price.png` 등)

- **리포트 저장:**
  - 전체 분석 결과(요약, 주요 통계, 그래프 파일 경로 포함)를 `eda_report.md` 파일로 저장
  - Markdown 문서는 다음 구조를 따름:
    ```
    # 1. 데이터 불러오기
    ...
    # 2. 데이터 개요
    ...
    # 3. 기술통계 요약
    ...
    # 4. 결측치 및 이상치 분석
    ...
    # 5. 변수 간 관계 분석
    ...
    # 6. 시각화 요약
    ![Correlation Heatmap](eda_images/correlation_heatmap.png)
    ...
    # 7. 주요 인사이트
    ...
    ```

- 사용 라이브러리: pandas, numpy, matplotlib, seaborn, os
- 데이터프레임 이름은 `df`로 통일
- 모든 분석 및 설명은 **한글로 작성**
- 실행 순서에 따라 완전히 재현 가능한 형태로 작성
