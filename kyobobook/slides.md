
# 데이터 분석 실무
## 교보문고 베스트셀러 데이터 탐색적 분석 개요
- 파이썬 기반의 데이터 스크래핑 및 전처리 과정을 거친 베스트셀러 데이터의 전반적인 특성을 심도 있게 분석함
- 2026년 1월 17일 수집된 최신 교보문고 데이터를 바탕으로 시장의 흐름과 도서 소비 트렌드를 파악함
- 데이터 탐색적 분석 과정을 통해 결측치 처리 및 이상치 탐지 등 데이터 품질 개선을 위한 필수 작업 수행함
- 실무에서 즉시 활용 가능한 데이터 시각화 기법과 통계적 해석 방법을 학습하여 데이터 분석 역량을 강화함
- 본 슬라이드 교육 자료를 통해 원본 데이터로부터 비즈니스 인사이트를 도출하는 일련의 과정을 상세히 제시함

::: notes
오늘 강의에서는 교보문고 베스트셀러 데이터를 활용한 실무형 데이터 분석 과정을 다룹니다. 데이터 수집부터 정제, 시각화까지의 전 과정을 실제 리포트를 기반으로 학습할 예정입니다.
:::

# 데이터 수집
## 베스트셀러 API 데이터 로드 및 환경 설정
- 교보문고 서버로부터 API 방식을 통해 수집된 베스트셀러 원본 데이터를 분석 시스템에 성공적으로 로드 완료함
- 데이터 분석의 정확성을 기하기 위해 인코딩 방식의 적정성을 검토하고 파이썬 판다스 라이브러리를 활용하여 파싱함
- 원본 데이터의 손상을 방지하기 위해 로우 데이터를 별도 보관하고 복제된 본본을 분석 대상으로 지정하여 안전성 확보함
- 분석에 필요한 기초 라이브러리인 넘파이와 판다스 그리고 시각화를 위한 맷플롯립 라이브러리 환경 설정을 마침
- 수집된 데이터의 시간적 범위를 확인하여 2026년 1월 현재의 시장 상황을 반영하는지 철저하게 검증 과정을 거침

::: notes
데이터 분석의 첫 단계는 정확한 데이터를 불러오는 것입니다. API를 통해 수집된 데이터의 인코딩 문제를 해결하고 분석 환경을 구축하는 과정을 설명합니다.
:::

- 도서의 순위를 나타내는 랭크 변수부터 제목과 저자 그리고 출판일 등 도서 식별을 위한 핵심 정보가 존재함
- 판매 가격과 실제 할인가 그리고 리뷰 점수와 리뷰 수 등 소비자의 반응을 수치화할 수 있는 변수를 포함함
- 카테고리 분류 정보를 통해 도서의 장르별 분포를 파악할 수 있으며 웹 링크 주소로 실제 상세 페이지 확인 가능함
- 데이터 수집 시점을 기록한 크롤링 일시 변수를 통해 시계열적 관점에서의 순위 변화 추적을 위한 기초 마련함

# 변수 명세
## 데이터 컬럼별 속성 및 데이터 타입 정의
- 각 컬럼의 데이터 타입을 확인하여 수치형 데이터와 범주형 데이터를 명확히 분류하고 분석 기법을 선정함
- 랭크와 가격 그리고 리뷰 수 등은 정수형 타입으로 처리하여 통계 연산 및 수치 비교 분석이 가능하도록 설정함
- 도서 제목과 저자 그리고 카테고리 정보는 객체 타입으로 처리하여 텍스트 분석 및 그룹화 작업에 활용함
- 출판일 데이터는 실수형 타입으로 로드되었으나 향후 날짜 형식으로의 변환을 고려하여 데이터 유효성을 검사함
- 리뷰 점수는 소수점 단위를 포함하는 실수형 타입으로 관리하여 사용자 만족도의 미세한 차이를 정밀하게 분석함

| 컬럼명 | 데이터 타입 | 주요 내용 | 비고 |
|---|---|---|---|
| rank | int64 | 도서 판매 순위 | 핵심 지표 |
| title | object | 도서 제목 정보 | 식별자 |
| price | int64 | 도서 정가 | 수치형 |
| category | object | 도서 장르 분류 | 범주형 |

::: notes
데이터의 성격에 맞는 분석을 위해 각 컬럼의 데이터 타입을 정의합니다. 숫자 데이터와 문자 데이터를 구분하는 작업은 이후 통계 분석의 기초가 됩니다.
:::

# 데이터 품질
## 결측치 발생 현황 파악 및 분포 분석
- 전체 500건의 데이터 중 일부 컬럼에서 결측치가 발견되어 데이터 완전성을 확보하기 위한 전수 조사를 실시함
- 카테고리 컬럼에서 19건의 결측치가 발생하여 도서 분류 체계의 누락 여부를 확인하고 보완 대책을 수립함
- 저자와 출판사 그리고 출판일 정보에서도 극소수의 결측치가 확인되어 원본 소스의 데이터 제공 상태를 점검함
- 결측치가 발생한 행의 특성을 분석하여 특정 카테고리나 출판사에서 집중적으로 발생하는지 상관관계를 검토함
- 유효한 통계 결과 도출을 위해 결측치 처리 전략으로 삭제 또는 대체값 삽입 등의 방법론을 신중히 선택함

::: notes
데이터 품질 관리의 핵심인 결측치를 확인합니다. 특히 카테고리 정보의 누락은 장르 분석 시 오차를 발생시킬 수 있으므로 주의 깊게 살펴봐야 합니다.
:::

# 시각화 분석
## 결측치 분포 시각화 및 데이터 누락 지점 확인
- 데이터 세트 전체의 결측치 발생 지점을 히트맵 형식의 시각화 자료를 통해 한눈에 파악 가능하도록 구현함
- 흰색으로 표시된 결측치 영역을 통해 데이터 수집 과정에서 누락된 정보의 위치를 직관적으로 식별 완료함
- ![Missing Values](eda_images/missing_values.png)
- 특정 변수들 사이에서 결측치가 동시에 발생하는 패턴이 있는지 시각적 도구를 활용하여 면밀히 분석함
- 결측치 시각화 결과는 데이터 정제 단계에서 우선적으로 처리해야 할 대상 목록을 선정하는 핵심 근거가 됨

::: notes
텍스트로 된 결측치 정보를 이미지로 변환하여 확인합니다. 시각화를 통해 데이터의 구멍이 어디에 있는지 직관적으로 파악할 수 있습니다.
:::

# 통계 요약
## 수치형 변수의 기술통계량 심층 분석
- 500위까지의 순위 데이터의 평균값은 약 251이며 표준편차를 통해 순위가 고르게 분포되어 있음을 확인함
- 도서 정가의 평균은 약 20,763원이며 최소 3,000원부터 최대 95,000원까지 넓은 가격 스펙트럼을 형성함
- 실제 판매 가격은 평균 18,754원으로 정가 대비 약 10퍼센트 수준의 할인이 보편적으로 적용됨을 파악함
- 리뷰 점수의 평균은 9.03점으로 매우 높게 형성되어 있으며 중앙값이 9.78점으로 대다수 도서가 고득점을 기록함
- 리뷰 수의 경우 최대 4,371건에 달하는 도서가 존재하여 베스트셀러 내에서도 독자 반응의 편차가 매우 큼

| 지표 | 가격(price) | 리뷰점수 | 리뷰수 |
|---|---|---|---|
| 평균 | 20,763 | 9.03 | 252 |
| 최대값 | 95,000 | 10.0 | 4,371 |
| 중앙값 | 19,000 | 9.78 | 65 |

::: notes
데이터의 수치적 특성을 요약합니다. 평균과 중앙값의 차이를 통해 데이터가 한쪽으로 쏠려 있는지 아니면 고르게 분포하는지 판단할 수 있습니다.
:::

# 가격 분석
## 도서 가격대의 분포 특성 및 시장 가격 형성
- 도서 정가는 주로 15,000원에서 25,000원 사이의 가격대에 가장 집중적으로 형성되어 있음을 시각적으로 확인함
- 50,000원을 초과하는 고가의 도서는 전문 서적이나 세트 구성 상품으로 추정되며 베스트셀러 중 소수 비중임
- ![Distribution price](eda_images/dist_price.png)
- 최저가 도서 그룹과 최고가 도서 그룹 사이의 격차를 분석하여 소비자들의 가격 저항선을 간접적으로 유추함
- 도서 가격의 분포 곡선이 오른쪽으로 긴 꼬리를 가진 형태를 보이며 일부 고가 도서가 평균치를 상승시킴

::: notes
도서 시장의 가격 형성대를 분석합니다. 대부분의 베스트셀러가 어떤 가격대를 형성하고 있는지 파악하여 가격 전략 수립에 참고할 수 있습니다.
:::

# 할인 정책
## 판매 가격 분석을 통한 도서 할인 트렌드 파악
- 대다수의 베스트셀러 도서들이 정가 대비 일정 비율의 할인을 적용하여 독자들에게 구매 유인책을 제공함
- 판매 가격의 분포는 정가 분포와 유사한 흐름을 보이지만 전체적인 가격대가 하향 조정되어 그래프가 좌측 이동함
- ![Distribution sale_price](eda_images/dist_sale_price.png)
- 할인가 적용 후 10,000원 이하로 내려가는 실속형 도서군이 일정 부분 존재하여 시장의 틈새 수요를 반영함
- 도서 가격 안정화 정책 범위 내에서 출판사와 유통사가 취할 수 있는 최대 할인폭이 적용된 것으로 판단함

::: notes
실제 소비자가 구매하는 가격인 할인가를 분석합니다. 정가와 할인가의 차이를 통해 도서 시장의 일반적인 할인율을 확인할 수 있습니다.
:::

# 독자 반응
## 리뷰 점수 분포를 통한 도서 만족도 측정
- 리뷰 점수의 분포를 보면 9점 후반대에 압도적으로 많은 데이터가 밀집되어 있어 독자들의 긍정적 평가가 주류임
- 0점에서 5점 사이의 낮은 점수를 기록한 도서는 매우 희소하며 이는 베스트셀러 진입 도서의 기본 품질을 입증함
- ![Distribution review_score](eda_images/dist_review_score.png)
- 리뷰 점수의 중앙값이 평균보다 높게 형성되어 있어 소수의 낮은 점수가 평균을 깎아먹는 부의 왜곡 현상 발생함
- 만점에 가까운 평점을 받은 도서들의 리스트를 추출하여 독자들의 높은 지지를 받는 공통적인 요소를 분석함

::: notes
독자들의 만족도를 점수로 분석합니다. 대부분의 베스트셀러가 높은 점수를 받고 있는데, 이는 독자들의 검증을 거친 책들이 순위에 오르기 때문입니다.
:::

# 인기도 지표
## 리뷰 수 분포를 통한 대중적 관심도 분석
- 리뷰 수의 분포는 0건에 가까운 도서부터 수천 건에 이르는 도서까지 극심한 편차를 보이는 비대칭 구조임
- 대다수의 도서는 200건 미만의 리뷰를 보유하고 있으나 일부 메가 베스트셀러가 전체 리뷰 수를 견인함
- ![Distribution review_count](eda_images/dist_review_count.png)
- 리뷰 수가 1,000건을 상회하는 도서들은 장기 베스트셀러이거나 강력한 팬덤을 보유한 저자의 저서일 확률이 높음
- 신규 진입 도서의 경우 판매량에 비해 리뷰 수가 적게 나타나는 경향이 있어 순위와 리뷰의 시차를 고려함

::: notes
리뷰의 개수는 책의 대중적인 인기를 직접적으로 보여줍니다. 소수의 책들이 엄청난 양의 리뷰를 독점하고 있는 현상을 시각적으로 확인할 수 있습니다.
:::

# 순위 분포
## 베스트셀러 순위 데이터의 균등성 확인
- 1위부터 500위까지의 순위 데이터는 수집 범위 내에서 누락 없이 고르게 분포되어 데이터의 연속성을 증명함
- 순위 분포 그래프가 수평선에 가까운 형태를 보이는 것은 수집된 샘플이 순위 구간별로 편중되지 않았음을 의미함
- ![Distribution rank](eda_images/dist_rank.png)
- 데이터 수집 시점의 일시적인 순위 변동 가능성을 배제하기 위해 전체적인 분포의 안정성을 우선적으로 검토함
- 순위 변수는 다른 수치형 변수들과 결합하여 순위가 높을수록 가격이나 리뷰 수가 어떻게 변하는지 분석함

::: notes
수집된 데이터의 순위가 빠짐없이 잘 들어왔는지 확인합니다. 1위부터 500위까지 고르게 분포되어 있어야 통계적 왜곡이 발생하지 않습니다.
:::

# 출판 동향

# 이상치 탐지
## 상자 수염 그림(Boxplot)을 활용한 특이 데이터 식별
- 수치형 변수들에 대해 상자 수염 그림 분석을 실시하여 전체 데이터 흐름에서 벗어난 이상치 후보군을 추출함
- 가격 변수에서 상단 수염을 크게 벗어나는 고가 도서들은 일반적인 단행본의 범주를 넘어서는 특수 도서임
- 리뷰 수 변수에서 발견되는 극단적인 이상치들은 대중적 인지도가 매우 높은 이른바 슈퍼 베스트셀러에 해당함
- 리뷰 점수에서 하단에 위치한 이상치들은 독자들의 기대치에 미치지 못한 논란의 도서일 가능성이 있어 정밀 검토함
- 이상치를 단순히 제거하기보다 해당 데이터가 가지는 비즈니스적 의미를 해석하여 분석의 깊이를 더하고자 함

[이상치 식별 기준]
- IQR(사분위범위)의 1.5배 초과 지점
- 하단 이상치: Q1 - 1.5 * IQR
- 상단 이상치: Q3 + 1.5 * IQR

::: notes
평균적인 데이터 범위를 벗어나는 특별한 케이스들을 찾습니다. 이런 이상치들은 대박 상품이거나 혹은 데이터 오류일 수 있어 확인이 필요합니다.
:::

- 이러한 스테디셀러들은 리뷰 수가 오랜 시간 축적되어 신간 도서가 넘기 힘든 신뢰의 장벽을 구축하고 있음
- 가격 변동이 적고 평점이 안정적이며 학교나 기관의 추천 도서 목록에 포함되는 등의 외부 요인이 강력하게 작용함
- 데이터 분석 시 신간 위주의 분석에서 벗어나 이러한 장기 생존 도서들의 패턴을 별도로 분리하여 고찰함

::: notes
두 번째 사례는 오래된 책들의 생존 전략입니다. 나온 지 오래되었는데도 여전히 베스트셀러인 책들은 어떤 데이터적 특징을 갖는지 분석합니다.

# 분석 인사이트 1

- 카테고리 정보의 결측치 19건은 전체 분석의 정확도를 저해할 수 있으므로 추가 수집이나 수동 보정 작업이 필수적임
- 수치 데이터로 로드된 출판일 데이터를 시계열 타입으로 변환하여 월별 또는 분기별 출판 트렌드 분석으로 확장함
- 리뷰 점수의 상향 평준화 현상을 극복하기 위해 가중치를 부여한 새로운 만족도 지표 개발의 필요성을 절감함
- 가격 이상치로 분류된 도서 그룹을 제외한 일반 단행본 중심의 서브셋을 구축하여 보다 정교한 통계 모델 도출함
- 데이터 수집 과정에서의 로그 기록을 정밀 분석하여 향후 자동화 에이전트의 안정적인 운영을 위한 최적화 수행함

# 분석 인사이트 2
## 시장 트렌드 반영 및 비즈니스 의사결정 활용
- 최신 출판 도서의 베스트셀러 점유율이 압도적이므로 출판 마케팅의 화력을 초기 3개월에 집중하는 전략 제안함
- 특정 출판사의 시장 점유율 독점 현상을 분석하여 후발 주자들이 공략 가능한 틈새 카테고리를 발굴할 필요성 있음
- 리뷰 수와 순위 간의 높은 관련성을 바탕으로 독자들의 리뷰 작성을 독려하는 프로모션의 효용성을 데이터로 증명함
- 고가 도서의 베스트셀러 진입 사례를 분석하여 가격 저항선을 극복할 수 있는 프리미엄 콘텐츠 구성 전략 수립함
- 수집된 데이터의 시계열적 변화를 지속적으로 모니터링하여 유행의 변화 속도를 측정하고 재고 관리 효율성 증대함

# 기술적 고찰

- 로구루(Loguru) 라이브러리를 활용한 상세 로깅 체계를 구축하여 스크래핑 및 분석 과정의 오류를 실시간 추적함
- 판다스 라이브러리의 강력한 연산 성능을 활용하여 500건의 데이터를 밀리초 단위로 처리하고 통계량을 산출함
- 맷플롯립과 코리아나이즈 맷플롯립의 결합을 통해 한글 깨짐 현상 없는 고품질의 시각화 결과물을 안정적으로 생성함
- 자동화된 EDA 스크립트를 통해 새로운 데이터 유입 시 별도의 코드 수정 없이 즉시 리포트 생성이 가능한 구조임
- 향후 머신러닝 모델을 도입하여 과거 순위 데이터를 학습하고 미래의 베스트셀러 등극 가능성을 예측하는 기능 확장함

분석에 사용된 기술들을 돌아봅니다. 파이썬과 다양한 라이브러리들이 어떻게 유기적으로 작동하여 이 보고서를 만들어냈는지 설명합니다.

- 텍스트 마이닝 기법을 도입하여 도서 제목과 리뷰 내용에서 핵심 키워드를 추출하고 감성 분석 결과와 순위 연결함
- 외부 변수인 경제 지표나 사회적 이슈 데이터를 결합하여 특정 도서의 유행 원인을 다각도로 분석하는 모델 개발함
- 사용자 맞춤형 추천 엔진 개발을 위해 독자들의 구매 패턴 데이터를 확보하여 개인화된 베스트셀러 예측 서비스 제공함

# 결론 및 요약
## 데이터 기반의 도서 시장 통찰 및 교육 마무리
- 교보문고 베스트셀러 데이터 분석을 통해 시장의 주류 장르와 출판사들의 영향력 그리고 독자들의 반응을 정밀 확인함
- 탐색적 데이터 분석(EDA) 과정은 단순한 수치 확인을 넘어 비즈니스 기회를 발굴하는 필수적인 전제 조건임을 학습함
- 시각화와 통계 분석을 통해 데이터 속에 숨겨진 패턴을 발견하고 이를 논리적인 인사이트로 연결하는 역량을 배양함
- 본 교육 자료에서 다룬 분석 기법들을 실무의 다양한 영역에 응용하여 데이터 중심의 의사결정 체계를 구축하길 권장함
- 데이터는 거짓말을 하지 않으며 올바른 분석 도구와 시각을 갖춘다면 복잡한 시장 상황에서도 명확한 해답을 찾을 수 있음

::: notes
전체 내용을 요약하며 강의를 마칩니다. 데이터를 통해 시장을 보는 눈을 기르고, 이를 실제 업무에 어떻게 적용할지 마지막으로 강조합니다.
:::
